notmarkdown

Community powered collection of markup libraries, for beginners and experts.

What is it?

notmarkdown is an opensource project with a purpose. It welcome beginners to contribute to learn about code and opensource, and warmly welcomes pros to add a sense of power and flair to the project capabilities while teaching beginners about what to do.

Besides the initiative, what is it? notmarkdown is a collection of highly flexible markup libraries created by the notmarkdown team, beginners learning how to contribute, and professionals who want to tackle a challenge and push the boundaries of markup functionality. The structure of the scripts creates oppertunity for the library to be as simple as "Hello, world!" or as complex as retraining deep learning neurons in Python. Everyone is invited to create a library that helps themselves and others, through syntax that fits the project.

Bare-bones notmarkdown html 
<html>
 <body>
  <!-- Input -->
  <p id="nmd-in">~/Hello, everyone/~. _/This/_ is ::/notmarkdown/::</p>
  <!-- Output -->
  <p id="nmd-out"></p>
  <!-- Standard notmarkdown script -->
  <script src="nmd.js"></script>
  <!-- Activate notmarkdown -->
  <script>parse()</script>
 </body>
</html>
Used snippet from standard library 
function parse() {
 text = document.getElementById("nmd-in").innerHTML
 output = text
  .replace(new RegExp("::/", "g"), "<strong>")
  .replace(new RegExp("/::", "g"), "</strong>")
  .replace(new RegExp("_/", "g"), "<u>")
  .replace(new RegExp("/_", "g"), "</u>")
  .replace(new RegExp("~/", "g"), "<p style='display:inline;font-family:cursive'>")
  .replace(new RegExp("/~", "g"), "</p>")
 document.getElementById("nmd-out").innerHTML = output
 console.log(output)
}
